## 功能

短时间的倒计时，支持 `1~60` 分钟的倒计时。午睡闹钟，主要是中午睡觉用的。

**注意！由于 Android 系统会在息屏后进入休眠状态，倒计时会因此变得不准确。不过，如果没有息屏，只是将应用转入后台，那么这是没有影响的。**
